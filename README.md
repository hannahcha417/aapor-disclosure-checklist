# AAPOR Disclosure Checklist for Use of AI in Surveys

Initial implementation of the AAPOR Disclosure Checklist for the Use of AI in Surveys

## Development Setup

Follow these steps to run this codebase:

### Install required libraries

Open a terminal in the root directory and run `npm install`.

### Set .env
Set .env with the necessary supabase key and project URL.

### Run the app

You can run the app by running the following command in your terminal:
`npm run dev`.
This will provide a link to a locally hosted server (e.g. http://localhost:5173/), which you can follow to view the interface.
